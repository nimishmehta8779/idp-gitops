# CAIPE Prompt Library

This prompt library is organized by persona to assist developers and operators in using CAIPE.

## Persona: Developer

### GitHub Operations
- "List open pull requests in nimishmehta8779/idp-gitops"
- "What is the status of the last workflow run in team-alpha-infra?"
- "Get the contents of the file catalog-info.yaml in nimishmehta8779/idp-platform"

### ArgoCD Status
- "Check the sync state of argocd-server application"

## Persona: Platform Engineer

### Kubernetes Inspection
- "List all namespaces in the cluster"
- "Show me all pods in the crossplane-system namespace"
- "Get the recent events in the clusters-dev namespace"

### Cluster Infrastructure
- "List all EKS clusters registered in the cluster"
- "Get the details and status of the managed resources"

## Persona: Incident Response
